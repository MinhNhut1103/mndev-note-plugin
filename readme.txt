Plugin để note lại các thông tin code tính năng website Mndev, cân nhắc kỹ trước khi xóa

== Description ==

Mndev Plugin là một plugin quản lý ghi chú chuyên biệt cho website Mndev. Plugin này cho phép bạn:

* Tạo, chỉnh sửa và xóa ghi chú một cách dễ dàng
* Lưu trữ thông tin về code đã viết, ý tưởng và các ghi chú quan trọng
* Giao diện đẹp mắt và trực quan
* Hiệu ứng mượt mà và thân thiện với người dùng
* Tìm kiếm và quản lý ghi chú hiệu quả

Plugin này đặc biệt hữu ích cho các nhà phát triển WordPress muốn ghi lại các thay đổi code, các ý tưởng mới hoặc các thông tin quan trọng trong quá trình làm việc.

== Features ==

* **Quản lý ghi chú đầy đủ**: Thêm, sửa, xóa ghi chú dễ dàng
* **Giao diện đẹp mắt**: Thiết kế hiện đại, trực quan và dễ sử dụng
* **Hiệu ứng mượt mà**: Các chuyển động và hiệu ứng đẹp mắt
* **Responsive**: Hoạt động tốt trên mọi kích thước màn hình
* **Bảo mật**: Chỉ người dùng có quyền quản lý mới có thể truy cập
* **Tối ưu hóa**: Tải nhanh và không ảnh hưởng đến hiệu suất website

== Installation ==

1. Tải plugin lên thư mục `/wp-content/plugins/` hoặc cài đặt trực tiếp từ WordPress admin
2. Kích hoạt plugin trong mục "Plugins" trong WordPress admin
3. Truy cập "Mndev Notes" trong menu admin để bắt đầu sử dụng
